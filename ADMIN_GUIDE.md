После входа вы увидите главную панель с статистикой:

## 🛠️ Управление продуктами

### Просмотр списка продуктов

1. Нажмите **"Продукты"** в боковом меню
2. Вы увидите таблицу со всеми товарами:
   - Название
   - Категория
   - Марка
   - ГОСТ
   - Цена
   - Статус наличия
   - Количество просмотров

### Создание нового продукта

1. Нажмите кнопку **"Создать"**
2. Заполните обязательные поля:
   - **Name (EN)** - название на английском
   - **Название (RU)** - название на русском
   - **Slug** - URL-адрес (например: `steel-pipe-76x5`)
   - **Категория** - выберите из списка

3. Заполните характеристики товара:
   - **Марка** - марка стали (например: 12ГФ)
   - **ГОСТ** - стандарт (например: ТУ 14-3Р-77-2004)
   - **Диаметр** - в мм
   - **Толщина** - в мм
   - **Длина** - в метрах
   - **Вес** - в кг

4. Укажите цену:
   - **Цена** - числовое значение
   - **Тип цены** - "Договорная" или "Фиксированная"

5. Установите флаги:
   - **В наличии** - есть ли товар на складе
   - **Рекомендуемый** - показывать на главной странице

6. Нажмите **"Сохранить"**

### Редактирование продукта

1. В списке продуктов нажмите на строку или кнопку **"Редактировать"**
2. Измените нужные поля
3. Нажмите **"Сохранить"**

### Удаление продукта

1. В списке продуктов нажмите кнопку **"Удалить"** (иконка корзины)
2. Подтвердите удаление

## 📁 Управление категориями

### Просмотр категорий

1. Нажмите **"Категории"** в боковом меню
2. Вы увидите список всех категорий с:
   - Названием
   - Slug
   - Иконкой (emoji)
   - Порядком сортировки
   - Родительской категорией

### Создание категории

1. Нажмите **"Создать"**
2. Заполните поля:
   - **Name (EN)** - название на английском
   - **Название (RU)** - название на русском
   - **Slug** - URL (например: `pipe-products`)
   - **Иконка** - emoji символ (🔧, 📄, ⚫, и т.д.)
   - **Порядок сортировки** - число для упорядочивания
   - **Родительская категория** - оставьте пустым для главной категории

3. Нажмите **"Сохранить"**

### Создание подкатегории

1. Создайте категорию как обычно
2. В поле **"Родительская категория"** выберите главную категорию
3. Сохраните

### Структура категорий

**Пример иерархии:**
```
Трубный прокат (родитель)
  ├── Стальная труба (подкатегория)
  ├── Профильная труба (подкатегория)
  └── Нержавеющая труба (подкатегория)
```

## 💡 Советы и лучшие практики

### Slug (URL-адреса)

- Используйте только латинские буквы, цифры и дефисы
- Делайте их короткими и понятными
- Примеры хороших slug:
  - `steel-pipe-76x5`
  - `seamless-pipe-57x6`
  - `pipe-products`

### Категории

- Создавайте сначала родительские категории
- Затем добавляйте подкатегории
- Используйте порядок сортировки (0, 1, 2...) для управления отображением
- Добавляйте emoji иконки для визуального различия

### Продукты

- Заполняйте все доступные поля для лучшего SEO
- Используйте понятные названия на русском
- Указывайте точные характеристики (ГОСТ, марка, размеры)
- Отмечайте популярные товары как "Рекомендуемые"

### Цены

- Для товаров с фиксированной ценой укажите число
- Для товаров "под заказ" выберите тип "Договорная"
- Цены отображаются в рублях на сайте

## 🆘 Часто задаваемые вопросы

### Как добавить изображения к товару?

В текущей версии поддерживаются URL изображений. Укажите прямую ссылку на изображение в поле `images`.

### Можно ли массово удалить товары?

Да, выберите несколько товаров галочками и используйте массовое удаление.

### Как изменить пароль администратора?

Пароль можно изменить через базу данных или создав новый эндпоинт в API.

### Что делать если забыл пароль?

Используйте стандартные учетные данные:
- Email: admin@beradinox.uz
- Password: admin123

Или сбросьте базу данных командой:
```bash
docker-compose exec backend npx prisma db seed
```
