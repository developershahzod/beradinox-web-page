## 🚨 Решение проблем

### Порты заняты

Измените порты в `docker-compose.yml`:
```yaml
frontend:
  ports:
    - "3002:80"  # вместо 3000
```

### База данных не запускается

```bash
docker-compose down -v
docker-compose up -d postgres
sleep 10
docker-compose up -d
```

### Админка не загружается

```bash
docker-compose logs admin
docker-compose restart admin
```

### Нет данных в админке

```bash
docker-compose exec backend npx prisma db seed
```

## 🎓 Обучающие материалы

### Как добавить товар

1. Админка → Продукты → Создать
2. Заполните:
   - Название (RU и EN)
   - Slug (URL)
   - Категорию
   - Характеристики (марка, ГОСТ, размеры)
   - Цену
3. Сохраните

### Как создать категорию

1. Админка → Категории → Создать
2. Укажите:
   - Название
   - Slug
   - Иконку (emoji)
   - Родителя (если подкатегория)
3. Сохраните

### Как изменить контакты

1. Админка → Настройки
2. Найдите нужную настройку
3. Кликните для редактирования
4. Измените значение
5. Сохраните
