## 🔄 How to Seed Database

### Option 1: Run Backend Locally
```bash
cd backend
npm run dev
```

The backend will automatically run migrations and seed on startup.

### Option 2: Manual Seed
```bash
cd backend
npx prisma db push
npx prisma db seed
```
